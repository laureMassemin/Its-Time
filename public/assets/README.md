# Assets - Images pour les marques et types de produits

## Structure des dossiers

### /types
Contient les images des types de produits (lipstick, foundation, etc.)

**Convention de nommage :** Utiliser le nom du type en minuscules avec underscores
- `lipstick.png` ou `lipstick.jpg`
- `foundation.png`
- `mascara.png`
- `eyeliner.png`
- `blush.png`
- `nail_polish.png`
- `eyeshadow.png`
- `bronzer.png`
- `lip_liner.png`
- `eyebrow.png`
- `concealer.png`
- `powder.png`
- `highlighter.png`
- `contour.png`

### /brands
Contient les logos des marques

**Convention de nommage :** Utiliser le nom de la marque en minuscules
- `maybelline.png` ou `maybelline.jpg`
- `colourpop.png`
- `nyx.png`
- etc.

## Format recommandé
- **Format :** PNG avec transparence ou JPG
- **Taille recommandée :** 
  - Types : 200x200px
  - Marques : 150x150px ou format logo original
- **Poids :** < 50KB par image pour optimiser le chargement

## Image par défaut
Si une image n'est pas trouvée, une image par défaut sera affichée.
